# Markdown을 방법을 정리
# 1. 글자 크기 조절 (제목, 부제목, 내용)

```
# 안녕하세요
## 안녕하세요
### 안녕하세요
#### 안녕하세요
```
> # 안녕하세요
> ## 안녕하세요
> ### 안녕하세요
> #### 안녕하세요

# 2. -, *, +와 들여 쓰기(-, *, + 상관없이 들여쓰기로 구분됨)

```
- 안녕하세요
  - 감사해요
    - 잘있어요
      - 다시만나요
```
> - 안녕하세요
>   - 감사해요
>     - 잘있어요
>       - 다시만나요

# 3. 인용하기

```
> 안녕하세요
>  > 감사해요
>  >  > 잘있어요
>  >  >  > 다시만나요
```
> 안녕하세요
>  > 감사해요
>  >  > 잘있어요
>  >  >  > 다시만나요

# 4. 블럭

```
        ```
        print("hello");
        ```
```
> ```
> print("hello");
> ```
